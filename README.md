# Non-Performing Loans (NPL) Analysis Dashboard

## Project Overview
This project showcases an interactive Microsoft Excel dashboard developed to support monitoring and analysis of non-performing loans (NPLs). The dashboard enables business users to track portfolio balances, collection performance, and key contributors across branches and risk classes.

The dashboard was designed for operational reporting and management decision-making, with a focus on data accuracy, clarity, and usability.

## Key Features
- Interactive KPI tracking for loan portfolio performance
- Currency-based filtering 
- Branch-level and risk-class analysis
- Dynamic charts highlighting top NPL contributors
- Automated weekly and monthly reporting
- User-friendly navigation using Excel controls and macros

## Tools Used
- Microsoft Excel (Advanced)
- PivotTables and PivotCharts
- Excel Formulas
- VBA / Macros
